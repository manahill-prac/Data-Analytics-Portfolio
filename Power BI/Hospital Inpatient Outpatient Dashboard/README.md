#  Hospital Patient Wait Time Analytics Dashboard

An interactive **Power BI dashboard** developed to analyze hospital patient wait list trends, monitor key healthcare KPIs, and provide actionable insights across inpatient and outpatient services.

---

## Overview

This dashboard enables healthcare stakeholders to monitor patient wait times, identify high-demand specialties, and analyze trends across multiple dimensions through interactive visualizations and drill-down capabilities.

---

## Dashboard Pages

### Summary
- Executive KPI overview
- Monthly wait list trends
- Case type distribution
- Age profile analysis
- Specialty performance
- Year-over-year comparison

### Detailed View
- Hierarchical patient wait list analysis
- Specialty-level breakdown
- Age profile segmentation
- Interactive filters
- Expandable matrix for detailed exploration

### Drilldown
- Specialty-wise total wait list
- High-level performance comparison
- Focused analytical view

## KPIs

| Metric | Description |
|---------|-------------|
| Total Wait List | Total number of patients on the waiting list |
| Latest Month Wait List | Current month's waiting list |
| Previous Year Wait List | Year-over-year comparison |
| Average Wait Time | Average waiting period |
| Median Wait Time | Median waiting period |

## Key Insights

- Outpatient cases account for the largest share of the waiting list.
- Adult patients contribute the highest patient volume across specialties.
- Several specialties consistently experience higher waiting times.
- Interactive drill-down enables detailed investigation by specialty, age group, and case type.
